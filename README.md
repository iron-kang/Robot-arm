# Robot-arm
## Robot arm simulation
![image](https://github.com/iron-kang/Robot-arm/blob/main/images/robot-arm-1.png)

## Requirement
* Qt Creator >= 4.5.1
* Armadillo library
```shell=
sudo apt install libopenblas-dev liblapack-dev libboost-dev
sudo apt-get install libarmadillo-dev
```
